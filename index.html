How to replace the placeholder:
  1. Upload your poster image to images/poster.jpg in this repository (or push it via Git).
  2. Supported types: .jpg, .jpeg, .png. For best results use a high-resolution poster (1200px+ wide).
  3. I added a CNAME for ampedupforwaramps.com. Make sure your domain DNS has the GitHub Pages A records (or an ALIAS/ANAME) pointing to GitHub.
